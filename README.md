# Biblioteca 2

Biblioteca 2 é a segunda versão de um sistema simples de biblioteca desenvolvido em Python com foco em programação orientada a objetos.
O projeto foi criado para praticar a modelagem de objetos e a interação entre classes, utilizando uma estrutura mais organizada do que a primeira versão.
Nesta versão, o sistema trabalha com três elementos principais: livros, usuários e empréstimos. A lógica implementada permite realizar empréstimos, verificar se um livro está disponível, registrar devoluções e impedir ações inválidas quando não há empréstimo ativo.

## Estrutura do projeto

- `livro.py` : representa os dados do livro
- `usuario.py` : representa os dados do usuário
- `emprestimo.py` : contém a lógica do empréstimo e da devolução
- `rodar.py` : arquivo principal para testes

## Funcionalidades

- criar livros
- criar usuários
- registrar empréstimos
- validar disponibilidade do livro
- registrar devoluções
- exibir mensagens de controle do sistema

## Conceitos aplicados

- classes e objetos
- associação entre objetos
- atributos e métodos
- separação em arquivos
- regras de negócio com POO

## Objetivo

Este projeto foi desenvolvido com o objetivo de reforçar a prática de Python e programação orientada a objetos por meio de uma segunda versão de um sistema de biblioteca, com estrutura mais clara e lógica mais consistente.

## Como executar

Abra o projeto no VS Code e execute o arquivo `rodar.py`.

## Tecnologias utilizadas

- Python
